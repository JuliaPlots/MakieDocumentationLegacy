barplot

AbstractPlotting.barplot — Function
barplot(x, y; kwargs...)

Plots a barplot; y defines the height. x and y should be 1 dimensional.

Attributes

Available attributes and their defaults for Combined{AbstractPlotting.barplot,T} where T are: 

  color        :black
  colormap     :viridis
  colorrange   AbstractPlotting.Automatic()
  direction    :y
  fillto       0.0
  marker       GeometryBasics.HyperRectangle{N,T} where T where N
  strokecolor  :white
  strokewidth  0
  visible      true
  width        AbstractPlotting.Automatic()
source

Examples

using GLMakie

xs = 1:0.2:10
ys = 0.5 .* sin.(xs)

barplot(xs, ys, color = :red, strokecolor = :black, strokewidth = 1)
barplot!(xs, ys .- 1, fillto = -1, color = xs, strokecolor = :black, strokewidth = 1)
current_figure()
