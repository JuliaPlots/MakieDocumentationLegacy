linesegments

AbstractPlotting.linesegments — Function
linesegments(positions)
linesegments(x, y)
linesegments(x, y, z)

Plots a line for each pair of points in (x, y, z), (x, y), or positions.

Attributes

Available attributes and their defaults for Combined{AbstractPlotting.linesegments,T} where T are: 

  ambient         Float32[0.55, 0.55, 0.55]
  color           :black
  colormap        :viridis
  diffuse         Float32[0.4, 0.4, 0.4]
  lightposition   :eyeposition
  linestyle       "nothing"
  linewidth       1.0
  nan_color       RGBA{Float32}(0.0f0,0.0f0,0.0f0,0.0f0)
  overdraw        false
  shininess       32.0f0
  specular        Float32[0.2, 0.2, 0.2]
  ssao            false
  transparency    false
  visible         true
source

Examples

using GLMakie

xs = 1:0.2:10
ys = sin.(xs)

linesegments(xs, ys)
linesegments!(xs, ys .- 1, linewidth = 5)
linesegments!(xs, ys .- 2, linewidth = LinRange(1, 10, length(xs)))
linesegments!(xs, ys .- 3, linewidth = 5, color = LinRange(1, 5, length(xs)))
current_figure()
